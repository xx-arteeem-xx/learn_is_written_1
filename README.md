# Криптографические задачи
Консольное приложение для решения криптографических задач на Python.

## Задачи

### 1. Шифр Вернама
- **Описание**: Шифрование трёх 8-битных чисел с использованием ключа методом XOR
- **Входные данные**: 3 числа открытого текста и 3 ключа
- **Процесс**: 
  - Перевод чисел в двоичный вид
  - Поразрядное XOR с ключом
  - Обратный перевод в десятичную систему

### 2. Линейный конгруэнтный генератор
- **Описание**: Генерация псевдослучайных чисел по формуле: `xₙ₊₁ = (a × xₙ + b) mod c`
- **Входные данные**: Параметры a, b, c, начальное значение x₀, количество чисел
- **Процесс**: Последовательное вычисление следующих значений

### 3. Генерация чисел RC4
- **Описание**: Генерация псевдослучайной последовательности с использованием алгоритма RC4
- **Входные данные**: Параметр n (определяет размер массива 2ⁿ), начальное состояние S
- **Процесс**:
  - Инициализация массива состояний
  - Итеративное обновление индексов i, j
  - Генерация выходных значений

### 4. Формирование перестановки RC4
- **Описание**: Инициализация начальной перестановки для алгоритма RC4
- **Входные данные**: Параметр n, длина ключа L, значения ключа K₀...Kₗ₋₁
- **Процесс**:
  - Начальное заполнение массива
  - Перемешивание с использованием ключа
  - Вывод промежуточных состояний
